Conceitos

O MyItCSS surge de uma ideia de arquitetura para se escrever o CSS. Ele não é uma biblioteca ou um framework, é uma ideia, um guia.

Ele é construído em cima da filosofia do Inverted Triangle CSS, o que lhe dará escalibilidade, fácil manutenção e organização no seu CSS.

Isso nos dá um CSS escalável, de fácil manutenção e de baixa complexidade

Como funciona / Configurações

Em ordem, nós escrevemos:

  • Configurações (Settings)
    As configurações gerais do MyIt, como as cores, espaçamentos, breakpoins e etc
  • Genéricos (Generic)
    Remover os defaults dos browsers
  • Elementos (Elements)
    Personalizar as tags para todo o projeto, exemplo: _images.scss está setando que toda imagem do projeto, por padrão, terá um "height: auto"
  • Objetos (Objects)
    Parecido com componentes, porém, mais complexos, tem o intuito de facilitar a construção de coisas que precisam de mais de uma tag. Todos vão começar com "_o-"
  • Componentes (Components)
    Classes para coisas que utilizam apenas 1 tag, sem hierarquia definida. Todos vão começar com "_c-"
  • Utilitários (Utilities)
    A melhor parte, o que vai nos poupar tempo para escrever, são as propriedades do CSS reduzidas a 1 classe cada. Todos vão começar com "_"
Sendo o triângulo com a ponta no Utilities, na parte de baixo e a base nas configurações, na parte de cima
